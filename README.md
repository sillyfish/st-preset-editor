# st-preset-editor

适用于编辑 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 预设JSON的小工具，在手机浏览器上改预设（电脑也可用）。

[点击直达编辑器](https://sillyfish.github.io/st-preset-editor/），或用浏览器打开 `st-preset-editor.html`。

界面内点右上角的 **“i”** 有使用说明。

## 特点

- **纯本地运行**：就是一个单独的 HTML 文件，用浏览器打开就能用，不用装东西、不用服务器。
- **不上传你的文件**：你导入的预设只在你自己的浏览器里处理，**不会发到任何服务器**，改完导出回本地就行。
- 只改 `prompts` 和 `prompt_order` 两部分，其余字段导出时**原样保留**。
- 稿纸风黑白界面、装订线开关、宏展开、搜索替换、Markdown 预览、撤回重做……都在里面。

## 关于访问统计

本站使用 GoatCounter 统计访问量，不使用 cookie，不采集个人信息。
